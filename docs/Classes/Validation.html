Validation

final class Validation

This class has the validation function required for this project.

vapor/validation is adopted as Infrastructure. This class was prepared because realworld’s production implementation (https://conduit.productionready.io/api) validation was an exhaustive check. At first, Validatable was used.

Extras

As an aside, I like vapor/validation. This is because there is a usage that is loosely coupled to vapor/vapor. If it is an add-in to vapor/vapor Package like fluent, I think it is better. Looking here, I expect it will probably be in the future.
